Локальный поисковик (версия 3)
План разработки
=========================

Структура работающего контейнера:
- база
  - таблица директорий
    - код директории
    - путь к директории
    - дата сканирования
  - таблица файлов
    - код файла
    - путь к файлу
    - дата сканирования
  - таблица индекса
    - код слова
    - код файла
  - таблица слов
    - код слова
    - слово
- крон
  - периодический запуск скрипта индексирования
- главный скрипт
  - создание объекта сервера
  - обработка поиска
- скрипт индексирования
- каталог с файлами индексирования
- каталог статических файлов для Flask
  - стили
- каталог шаблонов для Flask
  - базовый шаблонов
  - шаблон индекса

docker build --pull --rm -f "Dockerfile" -t lindex3:latest .
docker run -d -p80:5000 --mount type=bind,source=C:\Users\prog\Downloads,target=/app/static/files,readonly --name lindex3 lindex3
docker container ls
docker rm -f lindex3
docker exec -it lindex3 /bin/sh
docker image ls 
docker image prune
