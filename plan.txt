Локальный поисковик (версия 3)
План разработки
=========================

Структура работающего контейнера:
- база
  - dirs - таблица директорий
    - dir_path - путь к директории
    - время сканирования сканирования
    - время удачного сканирования
  - таблица файлов
    - путь к файлу
    - удачного обнаружения
  - таблица индекса
    - код слова
    - код файла
  - таблица слов
    - код слова
    - слово
- крон
  - периодический запуск скрипта индексирования
- главный скрипт
  - создание объекта сервера
  - обработка поиска
- скрипт индексирования
- каталог с файлами индексирования
- каталог статических файлов для Flask
  - стили
- каталог шаблонов для Flask
  - базовый шаблонов
  - шаблон индекса

+ первичный запуск (самая простейшая)
+ первая сборка
+ простое сканирование каталога
  + сканирование каталогов
  + сборка индекса
  + в крон
  + контроль в контейнере
+ вывод результатов
  + вывод общего списка
  + передача параметров поиска
  + подключение поиска по словам
- облагородить выдачу
  - посмотреть лекции по фреймворку
  - заголовок влево
  - поисковую строку на середину 
  - активировать при загрузке поисковую строку
  - найденное мельче (подсмотреть в гугле)
  - страницы больше и легче для нажатия мышкой
  - меню в точке справа сверху
  - страница о программе
- описание
  - ссылка на предыдущую версию в описании
